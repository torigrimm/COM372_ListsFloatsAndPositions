Lists and the Display Property

In the exercise section below is an unordered list with the id "list1." Add a style rule selecting this list to the stylesheet, and then add the properties described below and observe what happens.
  1. Set the "list-style" property to the value "circle inside"
  2. Set the list-style to "circle outside"
  3. Set the list-style to "square"
  4. Set the list-style to "none"
  5. border:2px red;
  6. padding: 10px;
  7. Set a padding-left property, AFTER the padding property, and set it to 40px. Notice what happens
  8. Remove the padding-left property and set the padding to 0.
We can also select the items inside the list by using the selector "#list1 li." This tells the browser to select all li tags inside the element with the id "list1." Add a rule with this selector to the style2.css file and try out the following properties (leave the css for the list where you left it for the last exercise)! Note how the appearance changes for each setting.
  1. Add a 1px thick black dashed border.
  2. padding: 10px;
  3. width: 200px;
  4. margin: 20 px;
  5. display: incline;
  6. p.inline { display: block; }
  7. Set the display property to "inline-block"
