Columns

Below the ruler are two divs, with ids testDiv1 and testDiv2. Let's make them into columns!! Try the following and see what happens:
  1. Float both divs right, by adding a float property with the value "right" to the css rules for each div in style2. What happens?
  2. Float both divs left, what happens?
  3. Float testDiv1 right and testDiv2 left. What happens?
  4. Float both divs to the left or to the right and set the size of testDiv1 to 60%. What happens?
  5. If we want two equally sized columns, what is the largest size setting in % we can set both divs to with these padding and margin settings?
  6. What happens if you resize your browser window after setting the divs to the max width setting you discovered in the previous step?
  7. Try some different padding and margin settings, and see how that affects the maximum width setting that you can use for your columns
  8. In the html source code for this page, copy testDiv1 and paste the copy below testDiv2. Change the id for this new div to testDiv3. Add a #testDiv3 section to the style rules. Now you have three columns to play with.
  9. Try to make a three column display with all three columns equally sized.
  10. Try to make a three column display with the far left column about 25% of the page width, and the other two splitting the remaining 75%
  11. Try to make a three column display with the far left column about 20% of the page width, the far right about 20%, and the middle column about 60%
